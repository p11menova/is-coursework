# Изображения категорий

В этой директории должны находиться PNG изображения для категорий новостей.

## Правила именования файлов:

- Имя файла должно соответствовать названию категории в нижнем регистре
- Пробелы заменяются на дефисы
- Расширение файла: `.png`

## Примеры:

Для категорий:
- "Политика" → `политика.png`
- "Спорт" → `спорт.png`
- "Технологии" → `технологии.png`
- "Бизнес" → `бизнес.png`
- "Культура" → `культура.png`
- "Наука" → `наука.png`
- "Здоровье" → `здоровье.png`

## Размеры изображений:

- Рекомендуемый размер: 400x200 пикселей
- Минимальный размер: 300x150 пикселей
- Формат: PNG (поддерживает прозрачность)

## Fallback

Если изображение для категории не найдено, будет показан цветовой блок старого стиля.

## Тестирование

Добавьте хотя бы одно изображение, чтобы проверить работу функционала.
